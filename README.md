# restaurant_springboot  
基于SSM+springboot+vue+MySQL实现的宿舍带饭管理系统  

事出由于作者一个不爱出门的室友，在为室友带饭后却迟迟收不到带饭钱的转账。一次两次三次，最后作者也忘记了为室友带过多少次饭？室友欠我多少钱？————无奈之下写下了这个网站。以此来处理那些总是不及时A钱的室友！！！  


# 运行效果如图
可以实现新增 编辑 删除等基础功能  
可以指定查询的列进行查询  
可以选择对应的“食神” 和 “带饭侠” 进行结账清算，计算出二者对应的债务关系
